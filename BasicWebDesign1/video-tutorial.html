How to Embed a Youtube Video into Your Website

1. First, open youtube and search for the video you would like to embed


2. Once you've clicked on the video, scroll down and find the 'share' button. Hit the share button to open up the menu and locate the feature that says 'embed'


3. After the embed button haas been clicked, it will open up to a code. Copy this code


4. Open up your website's html (this tutorial used GitHub) and paste the embed code between your body tags


5. Create a div around the embedded code and place the embedded code inside of the div. Adjust height and width as you wish

Now you have embedded the video into your website!
